SPI机制01-Java SPI
2020-11-29

1、什么是 Java SPI

SPI 全称（Service Provider Interface），JDK 提供的内置服务运行时发现机制，可以通过修改配置文件替换对应接口的具体实现。

2、如何使用

业务场景

比如网站需要登录，有多种不同的登录方式（微信、QQ、邮箱），Action 都是登录，抽象好接口后，只需要实现不同的登录方式即可，通过 SPI 动态加载，后面就算新增方式，也无需修改业务流程代码，对扩展开放。
